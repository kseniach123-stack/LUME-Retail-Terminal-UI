# LUME - Retail Checkout System

A simple checkout program for stores that works on iPad. Black and white, clean and easy to understand.

**Try it:** https://kseniach123-stack.github.io/LUME-Retail-Terminal-UI/

---

## What it does

- **Product Search** — Start typing product name, get instant suggestions
- **Shopping Cart** — Add items, program calculates the total
- **Barcode Scanning** — Use iPad camera to scan products
- **Sales History** — See which products were sold and when
- **Reports** — How much money was made, how many items, average sale

---

## How it works

### Project files
```
index.html          — main page
main.css           — styles (how it looks)
mobile.css         — settings for mobile devices
app.js             — program startup and data
ui.js              — what shows on screen
actions.js         — how everything works (logic)
assets/images/     — product photos
```

### Technical choices

#### 1. Clean JavaScript (no libraries)
- Program loads fast
- Small file size
- All data stays on iPad

#### 2. Data saved on iPad
When cashier closes the program, data stays in device memory. Next time you open it, everything is still there.

#### 3. Camera for barcode scanning
Program uses iPad camera. You can switch between front and back camera.

#### 4. Beep sound
When item is scanned, program makes a beep sound. Sound is generated by program, no separate files needed.

## Important to know

### What doesn't work yet
1. **Real barcode scanning** — Right now it's just simulation (random items)
2. **No internet** — Data is saved only on iPad, not in cloud
3. **No real login** — Anyone can click "Login"
4. **No detailed reports** — Only basic numbers
5. **Products are fixed** — You can't change the list

## How to run

### On your computer
```bash
# Download the project
git clone https://github.com/kseniach123-stack/LUME-Retail-Terminal-UI.git
cd LUME-Retail-Terminal-UI

# Open in browser
open index.html

# Or start local server
python -m http.server 8000
```

### On GitHub Pages
Already working here: https://kseniach123-stack.github.io/LUME-Retail-Terminal-UI/

To deploy your own version:
```bash
git push origin master:main
```
